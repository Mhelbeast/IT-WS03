# JobSeeker & MyBlog

A collection of projects built with vanilla PHP, CSS, and Docker.

## Projects

- **JobSeeker**  
  A job listing platform with features for posting jobs, browsing listings, account registration, and profile suitability analysis.

- **MyBlog**  
  A personal blog showcasing design work, reflections, and learning journey.

## Setup

```bash
git clone https://github.com/mhelveenserranoneust-prog/IT-WS03.git
cd IT-WS03
docker compose up --build
```

Access the application at:  
http://localhost:3000

## Tech Stack

- PHP  
- CSS  
- Docker
